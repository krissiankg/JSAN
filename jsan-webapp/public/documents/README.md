# Documents événement — JSAN

Les guides auteurs (PDF / Word) sont gérés via **Paramètres** du dashboard (`/dashboard/parametres`), réservé aux rôles **organisateur** et **super admin**.

Fichiers stockés dans Supabase Storage (`event-documents`), métadonnées dans `events_config.documents_config`.

Migrations : `010_documents_config.sql`, `011_event_documents_storage.sql`
